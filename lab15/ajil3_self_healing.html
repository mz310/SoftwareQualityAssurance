АЖИЛ 3: SELF-HEALING AUTOMATION ОЙЛГОЛТ

А. Анхны HTML UI элемент:

<button id="loginBtn">Login</button>

Уламжлалт Selenium код:

driver.find_element(By.ID, "loginBtn").click()

B. Эвдрэл үүсгэх - HTML өөрчлөлт:

<!-- Хуучин (эвдэрсэн) -->
<button id="loginBtn">Login</button>

<!-- Шинэ (өөрчлөгдсөн) -->
<button id="signinButton">Login</button>

Асуудал: ID өөрчлөгдсөн тул уламжлалт Selenium код эвдэрнэ!

C. AI санал болгосон Self-healing шийдлүүд:

1. Dynamic Locator Strategy (Олон locator-уудыг туршиж үзэх):

def find_element_with_fallback(driver):
    locators = [
        (By.ID, "loginBtn"),
        (By.ID, "signinButton"),
        (By.XPATH, "//button[contains(text(), 'Login')]"),
        (By.CSS_SELECTOR, "button.login-button"),
        (By.CLASS_NAME, "btn-login")
    ]
    for locator in locators:
        try:
            element = driver.find_element(*locator)
            if element.is_displayed():
                return element
        except NoSuchElementException:
            continue
    raise Exception("Login button not found with any locator")

2. Text-based Locator (Текст дээр суурилсан):

# Текст нь ихэвчлэн байнга хэвээр байдаг
driver.find_element(By.XPATH, "//button[text()='Login']").click()
# эсвэл
driver.find_element(By.XPATH, "//button[contains(text(), 'Login')]").click()

D. Self-healing зарчмын давуу тал:

  • Robustness: UI өөрчлөгдсөн ч тест ажиллана
  • Maintenance reduction: Locator-уудыг байнга засах шаардлагагүй
  • AI Intelligence: Хамгийн найдвартай locator-ийг автоматаар сонгоно
  • Confidence scoring: Locator-ийн найдвартай байдлыг үнэлнэ

E. Жишээ: Шинэ DOM-оос AI санал болгосон locator-ууд:

<button id="signinButton" class="btn-primary" data-testid="login-button">Login</button>

AI санал болгосон locator-ууд (priority дарааллаар):

  1. By.XPATH, "//button[text()='Login']" - 95% confidence (текст тогтвортой)
  2. By.CSS_SELECTOR, "button.btn-primary" - 80% confidence (class name)
  3. By.XPATH, "//button[@data-testid='login-button']" - 90% confidence (test attribute)
  4. By.ID, "signinButton" - 70% confidence (ID өөрчлөгдөж болно)

Дүгнэлт:

Self-healing automation нь AI-ийн тусламжтайгаар UI өөрчлөлтөд автоматаар дасан зохицож, тестийн найдвартай байдлыг нэмэгдүүлнэ. Энэ нь UI automation-ийн хамгийн том асуудлуудын нэг болох "brittle tests" (эмзэг тестүүд)-ийг шийдэхэд тусална.
